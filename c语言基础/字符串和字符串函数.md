# 一、表示字符串和字符串I/O

## 1. 在程序中定义字符串

### 1.1 字符串字面量（字符串常量）

> - 字符串字面量（字符串常量）
>   - 存储为字符数组，其后面会增加空字符"\0"
>   - 双括号括起来的内容
>   - 可以有转义字符
>   - 静态存储类别
>     - 只会被储存一次，存在于程序的整个生命周期

```c
// 定义示例 hello world
char greeting1[50]="hello ""world";
char greeting2[50]="hello world";
// 定义示例 hello "world"
char greeting2[50]="hello \"world\""
```

### 1.2 字符串数组和初始化

```c
// const 表示字符串不会改变
// 声明时指定数组大小,数组长度必须大于等于字符串长度+1（\0）
const char s1[10] = "hi";
// 未被利用的会被赋予空字符
const char s2[10] = {'h','i','\0'}
```
